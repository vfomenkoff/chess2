Создать репозиторий
git init

Добавить все измения для коммита
git add .

Создать коммит (Вместо test любое сообщение)
git commit -m "test"

Отправить коммиты на удаленный репозиторий
git push

Подтянуть коммиты с удаленного репозитория
git pull

Удалить последний коммит, но сохранить изменения
git reset HEAD~

Удалить последний коммит без сохранения изменений
git reset --hard HEAD~

Удалить последние 5 коммитов, но сохранить изменения
git reset HEAD~5

Создать ветку и перейти на неё (test - название ветки)
git checkout -b test

Просто перейти на ветку (test - название ветки)
git checkout test

Подтянуть изменения с другой ветки (например, с ветки master)
git merge master

Склонировать репозиторий
git clone https://github.com/vfomenkoff/chess2.git
